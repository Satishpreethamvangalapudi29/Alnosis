# AInosis – Medical Assistance Platform

AInosis is a web-based medical assistance platform designed to help users easily access essential healthcare services through a single interface. The platform connects users with doctors, laboratories, and medical stores, enabling them to find nearby healthcare facilities and interact with different medical service providers in a simple and organized way.

The main objective of this project is to simplify the process of accessing healthcare resources by integrating multiple medical service modules into one system. Users can explore healthcare options, locate nearby hospitals or pharmacies, and access services provided by doctors, laboratories, and medical stores.

The application is developed using Python with the Flask framework for backend functionality and HTML/CSS for the frontend interface. MongoDB is used as the database for managing user and service-related information. The system is structured into different modules such as User, Doctor, Laboratory, and Medical Store, each responsible for handling specific healthcare services within the platform.

AInosis demonstrates how modern web technologies can be used to build an integrated healthcare support system that improves accessibility, organization, and efficiency in accessing medical services.
